# 食谱功能说明

## 功能概述

基于 [cook.yunyoujun.cn](https://cook.yunyoujun.cn/) 的设计理念，在uni-app项目中实现了完整的食谱功能，包括食谱列表、搜索、分类、详情等核心功能。

## 页面结构

### 1. 食谱列表页 (`/pages/pagesServices/recipe/index`)
- **搜索功能**: 支持按食谱名称、描述、标签、食材等关键词搜索
- **分类筛选**: 中餐、西餐、日料、韩料、甜点、饮品、小吃等分类
- **热门推荐**: 展示热门食谱，包含烹饪时间和难度信息
- **食谱卡片**: 显示食谱图片、名称、描述、评分、标签等信息

### 2. 食谱详情页 (`/pages/pagesServices/recipe/detail`)
- **基本信息**: 食谱名称、描述、评分、烹饪时间、热量、份量、难度
- **食材清单**: 可勾选的食材列表，支持烹饪进度跟踪
- **烹饪步骤**: 详细的步骤说明，包含小贴士
- **营养信息**: 蛋白质、脂肪、碳水化合物等营养成分
- **烹饪小贴士**: 实用的烹饪技巧和建议

## 主要特性

### 🎯 用户体验
- **响应式设计**: 适配不同屏幕尺寸
- **流畅动画**: 点击反馈和过渡效果
- **直观导航**: 清晰的页面结构和导航路径

### 🔍 搜索功能
- **智能搜索**: 支持按食谱名称、描述、标签、食材等关键词搜索
- **实时筛选**: 分类和关键词组合筛选
- **搜索结果**: 空状态提示和加载更多功能

### 🥘 食材筛选功能
- **严格/模糊搜索**: 支持精确匹配和模糊匹配两种模式
- **多食材筛选**: 可同时选择多个食材进行筛选
- **包含模式**: 支持"包含所有食材"和"包含任意食材"两种筛选逻辑
- **匹配度显示**: 实时显示食谱与选择食材的匹配百分比
- **动态筛选**: 食材筛选与分类筛选、关键词搜索智能组合

### 📱 移动端优化
- **触摸友好**: 适合手指操作的按钮尺寸
- **滚动优化**: 横向滚动的分类标签
- **图片优化**: 合适的图片尺寸和加载模式

## 使用方法

### 1. 从首页进入
- 点击首页"食谱"快捷入口
- 直接跳转到食谱列表页

### 2. 浏览食谱
- 使用分类标签筛选不同菜系
- 查看热门推荐食谱
- 浏览完整的食谱列表

### 3. 搜索食谱
- 在搜索框输入关键词
- 支持食谱名称、食材、做法等搜索
- 实时显示搜索结果

### 4. 食材筛选
- **选择搜索模式**: 严格模式（精确匹配）或模糊模式（包含匹配）
- **添加食材**: 在食材输入框中输入食材名称，点击"添加"按钮
- **筛选逻辑**: 选择"包含所有食材"（AND逻辑）或"包含任意食材"（OR逻辑）
- **匹配度显示**: 开启"显示匹配度"可查看每个食谱的食材匹配百分比
- **清空筛选**: 点击"清空全部"可快速清除所有已选食材

### 5. 查看详情
- 点击食谱卡片进入详情页
- 查看完整的烹饪信息
- 勾选食材清单跟踪进度
