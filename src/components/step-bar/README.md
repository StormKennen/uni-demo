### 属性说明

| 属性          | 类型               | 描述                                   |
|---------------|--------------------|----------------------------------------|
| `data`       | `Array`            | 步骤数据，数组形式，每个对象包含一个 `title` 字段，表示步骤的名称。 |
| `maxStep`    | `Number`           | 最大步骤索引，表示用户可以点击的最大步骤。 |
| `currentStep`| `Number`           | 当前步骤索引，表示当前选中的步骤，支持双向绑定。 |

### 事件说明

| 事件            | 描述                                   |
|-----------------|----------------------------------------|
| `@handle-click` | 当用户点击步骤时触发，传递当前步骤索引作为参数。 |


### 组件使用

```html{0}
<StepBar :data="steps" :maxStep="maxStep" v-model:currentStep="currentStep" @handle-click="handleClick"/>
```
